# 西湖景区实时客流看板

本项目是浙江树人学院《人工智能》课程大作业。

## 项目简介

西湖景区实时客流看板是一个基于 Web 技术构建的可视化系统，整合了高德地图 API 与 ECharts 可视化库，模拟生成西湖景区各核心景点的实时客流数据，并以直观的方式呈现给用户。系统不仅展示实时客流数据，还提供了客流趋势分析、景点拥挤度评估、游览建议等功能，帮助用户全面了解西湖景区的客流分布情况。

## 技术栈

- **前端基础**：HTML5、CSS3、JavaScript（ES6+）
- **地图可视化**：高德地图 AMap API 2.0
- **数据可视化**：ECharts 5.4.3
- **样式增强**：CSS3 动画、弹性布局、响应式设计

## 核心功能

### 1. 实时客流监控

- 展示西湖 15 个核心景点（断桥、苏堤、雷峰塔、灵隐寺等）的实时客流数据
- 统计总游客数、拥挤景点数、畅通景点数，实时更新数据时间戳
- 以不同颜色标记景点拥挤度（空闲 / 舒适 / 较挤 / 拥挤），直观展示客流状态

### 2. 地图可视化

- 基于高德地图构建西湖景区地图底图，夜间模式提升视觉体验
- 自定义景点标记样式，标记大小随客流密度动态变化
- 标记 hover 动效、点击弹窗展示景点核心数据，增强交互体验

### 3. 景点详情分析

- 点击景点可展开详情面板，展示当前人流、承载上限、饱和度、今日峰值等核心指标
- 生成过去 24 小时客流趋势图表，标注早晚高峰时段、客流峰值
- 分析 1 小时客流变化趋势（上升 / 下降 / 平稳），提供游览建议时段

### 4. 智能交互功能

- 智能导游聊天面板，支持快速查询（如 "预测断桥""路线规划"）
- 基于时间因子与景点热度的客流数据模拟，贴近真实游览规律
- 响应式布局，适配不同屏幕尺寸的显示效果

### 5. 数据模拟逻辑

- 结合时段（早 6 点 - 晚 23 点）动态调整客流基础因子
- 基于景点自身承载量与热门程度生成差异化客流数据
- 模拟 24 小时客流历史数据，还原真实的客流变化趋势

## 运行方式

1. 克隆 / 下载本项目代码到本地
2. 确保本地有浏览器环境（Chrome/Firefox/Edge 等现代浏览器）
3. 直接打开 `xihu-dashboard.html` 文件即可运行
4. 注：地图功能需要网络连接，若高德地图 API 密钥过期，需替换为自己的密钥

## 项目特色

1. **视觉体验**：深色背景搭配渐变色系，符合数据可视化场景的视觉审美
2. **交互设计**：丰富的动效（标记缩放、波纹动画、面板滑入）提升用户体验
3. **数据驱动**：基于时间与热度的客流模拟算法，数据更贴近真实场景
4. **实用性**：提供游览建议、客流趋势分析，满足游客核心需求
